Hexo butterfly 自定义文章封面 && 主页顶部图片更改

问题来源：在给文章加封面后，实际网页中文章顶部也出现了封面图片

  1. 再主页中显示文章封面但是再文章页面不显示封面 参考：https://blog.csdn.net/qq_43857095/article/details/108306164
  2. 隐藏文章 参考：https://www.cnblogs.com/yangstar/articles/16690342.html#:~:text=%E5%AE%89%E8%A3%85%20%E5%9C%A8%E7%AB%99%E7%82%B9%E6%A0%B9%E7%9B%AE%E5%BD%95%E4%B8%8B%E6%89%A7%E8%A1%8C%20npm,install%20hexo-hide-posts%20--save
    需要安装hexo插件npm install hexo-hide-posts --save
    配置_config.yml 设置 filter: secret 之后，你就可以在 front-matter 中使用 secret: true 来隐藏文章了。
    使用
  • 你可以在命令行运行 hexo hidden:list 来获取当前所有的已隐藏文章列表。
  1. 一个文章加多个tags 多标签写法，这2种都是一样的效果，用哪个都可以
  2. 冷知识 使用hexo查看本地访问量数目回不对，上传后正确
  3. 一个文章加多个categories

在Typora中的图片设置

由于本博客搭建在服务器上，所以图像文件上传至服务器而不使用图床

  1. 根据多次尝试有以下方案
  • 上传至GitHub作为图床，再使用picgo与本地typora连接，缺点就是访问太慢需要VPN。设置的时候有问题，按网上教程配置好之后，有问题可以尝试一下将图床设置改为设为默认设置
  • 使用其他图床，需付费，访问速度快但是要付费，长期发展不合适。
